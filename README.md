# WEEK 1 - Security Assessment of Web Application

## Objective

The objective of this task was to perform a security assessment of a web application and identify common vulnerabilities using manual testing and OWASP ZAP.

## Application Used

OWASP NodeGoat (RetireEasy)

## Tools Used

* Node.js
* MongoDB
* Google Chrome
* OWASP ZAP 2.17.0

## Security Tests Performed

1. Cross-Site Scripting (XSS) Testing
2. SQL Injection Testing
3. Weak Password Storage Analysis
4. Security Misconfiguration Assessment
5. OWASP ZAP Automated Vulnerability Scan

## Findings

* Cross Site Scripting (Reflected) detected by OWASP ZAP
* Weak Password Storage identified
* MongoDB Security Misconfiguration identified
* Missing Security Headers detected
* Vulnerable JavaScript Library detected
